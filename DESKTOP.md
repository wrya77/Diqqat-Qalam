# دقة قلم — تطبيق سطح المكتب (Electron)

نسخة سطح المكتب تشغّل **نفس** التطبيق محلياً على حاسبتك:

- ✅ **يعمل بلا إنترنت** — يُقلِع الخادم داخلياً ويفتح مباشرةً بلا تسجيل دخول (تخزين على ملفات الجهاز).
- ✅ **تحكم USB/Serial بالآلة** — اتصال مباشر بـ GRBL/Mach عبر المنفذ التسلسلي (حزمة `serialport`)، وهو ما لا يتيحه المتصفح.
- ✅ نفس مولّد G-Code والأدوات والمحاكاة الموجودة في النسخة السحابية.

## التشغيل أثناء التطوير

```bash
npm install        # يثبّت electron + serialport (يُنزَّل electron مرة واحدة ~100MB)
npm run electron   # يفتح نافذة التطبيق
```

## بناء ملف التثبيت (Windows)

```bash
npm run dist:win
```

تُنتَج الملفات في مجلد `dist/`:

| الملف | الوصف |
|------|-------|
| `DiqqatQalam-Setup-2.0.0.exe`    | مُثبِّت (اختصار سطح المكتب + قائمة ابدأ، يمكن تغيير مسار التثبيت) |
| `DiqqatQalam-Portable-2.0.0.exe` | نسخة محمولة تعمل دون تثبيت (انسخها على USB وشغّلها مباشرة) |

## كيف يعمل

`electron/main.js` يضبط الوضع المحلي (يُفرِّغ متغيرات Supabase فلا يطلب تسجيل دخول)،
ثم يُحمّل `server.js` الذي يبدأ الاستماع على منفذ محلي (افتراضياً `38217`)،
وتفتح النافذة على `/app`. التحكم بالآلة يمرّ عبر نقاط `/api/cnc/*` التي تستخدم
`serialport` داخل عملية Node نفسها — لا حاجة لأي إعداد إضافي.

> ملاحظة: `serialport` تبعية اختيارية مبنية على N-API، فتعمل ثنائيّتها المُسبَقة مع
> إصدار Electron دون إعادة بناء. إن تعذّر تحميلها يعود الاتصال تلقائياً لوضع TCP فقط
> (يبقى توليد G-Code يعمل بالكامل).

## الأيقونة
